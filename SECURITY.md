# Security Policy

## Supported Versions

We will provide security fixes for the latest version of Orian Builder and encourage users through auto-updates to use the latest version of the app.

## Reporting a Vulnerability

Please file security vulnerabilities by opening a [GitHub Security Advisory](https://github.com/srideep47/OrianBuilder/security/advisories/new). Do not file security vulnerabilities as regular issues, as the information could be used to exploit users before a fix is available.
